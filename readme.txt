Hướng dẫn cài đặt
1. cài đặt xampp
2. cài đặt composer
3. import database (file db_todolist.sql trong forder root)
4. sửa lại file DB.php theo máy mình cài đặt
5. chạy link "http://localhost/Todolist_MVC/index.php?controller=works" xem kết quả :)

note: chạy lệnh "composer update" để cập nhập lại thư viện :)
